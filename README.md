# ERP Informative Website

A responsive Next.js website for explaining ERP software, modules, benefits, implementation steps, and demo requests.

## Features

- Tech-inspired ERP landing page
- English and Bangla language modes
- Responsive hamburger navigation
- Demo request form section
- SEO-friendly metadata
- Optimized logo and remote images

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
